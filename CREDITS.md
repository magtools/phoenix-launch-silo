# Credits

## Legacy Warp Project

The original `warp-engine` project was created to simplify local development environments for PHP applications.

That original repository was later abandoned, and this fork continued its maintenance before evolving into a broader orchestration direction.

## Stewardship Timeline

- Summa Solutions maintained the original Warp project until April 2021.
- Fork maintenance and ongoing development have been led by Martin Gonzalez since 2021.

Initial project authors:

- [Matias Anoniz](https://github.com/yosoyfunes/)
- [Sebastian Arrubia](https://github.com/sarrubia/)
